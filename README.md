# Sudoku Solver App

A full-stack web app to solve Sudoku puzzles using traditional and AI-based solving.

## Backend (Spring Boot)
- Endpoint: `POST /api/sudoku/solve`
- Endpoint: `POST /api/sudoku/ai-solve`

## Frontend (React + TailwindCSS)
- Input Sudoku board
- Solve via backend
